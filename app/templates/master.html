Disaster Response Project

Analyzing message data for disaster response

As part of my data science studies at Udacity, I built this API from the analysis and construction of a machine learning model from disaster data communications from different sources such as news, social media or emergency calls, with the purpose of directing the results to the agencies in charge of immediate attention.

The data set used was provided by Figure Eight, and consists of more than 26,000 messages classified in 36 different categories.
